# Security Policy

## 🛡️ Sovereign Security Promise
Kutumbly is built on the principle that **you own your data**. No cloud, no server, no compromise.

## 🔓 Reporting Vulnerabilities
We take the security and privacy of family data extremely seriously. If you discover a security vulnerability, please do NOT create a public issue. 

Instead, please send an encrypted report to:
**kutumbly@outlook.com**

Please include:
- A detailed description of the vulnerability.
- Steps to reproduce (Proof of Concept).
- Potential impact assessment.

## 🔒 Scope
The scope of this security policy covers:
- The Sovereign Gateway unlocking mechanism.
- AES-256-GCM encryption of `.kutumb` files.
- Local storage and session management.
- Biometric authentication integration.

## 🚫 Out of Scope
- External device security (OS-level breaches).
- User-managed PIN strength (we recommend complex Master PINs).
